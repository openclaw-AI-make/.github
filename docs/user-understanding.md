# 用户习惯理解系统

## 功能
让我能够理解你的简短表达，不用每次都详细解释

---

## 已学习的习惯

### 简洁指令
| 输入 | 理解 | 行动 |
|------|------|------|
| 开工/开工吧 | 开始执行 | 执行当前任务 |
| 就这样 | 同意方案 | 按计划执行 |
| 可以了 | 确认完成 | 结束任务 |
| 继续 | 延续操作 | 继续执行 |
| 算了 | 取消 | 停止任务 |

### 询问类
| 输入 | 理解 | 行动 |
|------|------|------|
| 怎么样 | 问状态 | 报告进度 |
| 有什么用 | 问作用 | 解释价值 |
| 怎么弄 | 问方法 | 说明步骤 |

### 优化类
| 输入 | 理解 | 行动 |
|------|------|------|
| 更快 | 要效率 | 优化速度 |
| 更简单 | 要简化 | 精简方案 |
| 自动化 | 要自动 | 写自动化脚本 |

---

## 工作类型快速识别

你说"安装"→ 我知道是技术实施
你说"写" → 我知道是内容创作  
你说"分析" → 我知道是数据分析
你说"赚钱" → 我知道是AI应用

---

## 使用示例

你可以说:
- "就这样执行" → 我懂你要确认
- "帮我配置下" → 我知道是技术实施
- "怎么弄" → 我会详细说明步骤

---

## 我会持续学习

你的每次反馈我都会记住:
- 说"不对" → 我记住这个方向不对
- 说"可以了" → 我记住这种方式可行
- 问"这个干嘛用的" → 我记住你在乎作用说明

让我越来越懂你。
